## Shell variables expansions

### Bash scripting involving variable declaration, shell expansions and more.

---

0. Create a script that creates an alias.

- Name <mark>`ls`</mark>
- Value: <mark>`rm *`</mark>
  > **Solution**
  >
  > _file_ `./0-alias` (executable)
  >
  > ```
  > #!/bin/bash
  > alias ls="rm *"
  > ```
  >
  > 2 lines
